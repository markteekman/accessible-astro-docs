## Usage
Accessible Astro exports two {%component%}-related components:
* `PartA`: The wrapper for an {%component%}
* `PartB`: The component that defines a single {%component%} item

### Props
#### `PartA` Props
`PartA` does not have any props. It accepts one or more `PartB` component(s) in the default slot.
#### `PartB` Props
* `header`
  * **Description:** The title to use for the accordion item
  * **Type**: `string`
  * **Default:** `"Default title"`
