`Media` is a component that displays `<img>` tags. It has a default empty `alt` tag which is required for non-decorative images. If the image is descriptive, for example, when there is no complementary text, then a descriptive `alt` tag must be used. The `Media` component also utilizes the `loading="lazy"` tag to optimize browser performance.

[Usage guidance](https://www.w3.org/WAI/tutorials/images/decorative/)

## Accessibility features
* Sets a default empty `alt` tag, which is required for non-descriptive images

## Usage
Accessible Astro exports one media-related components:
* `Media`: The image Component with a default `alt` tag

### Props
#### `Media` Props
* `alt`
  * **Description:** Descriptive text to use as the `alt` attribute of the image
  * **Type**: `string`
  * **Default:** `''`
* `class`
  * **Description:** One or more class names, as a string, to be applied to the `img` tag
  * **Type**: `string`
  * **Default:** `undefined`
* `src`
  * **Description:** The source URL of the image
  * **Type**: `string`
  * **Default:** `https://fakeimg.pl/640x360`

## Style customizations
The `Media` component does not have any unique classes. Style it like other `img` tags.
